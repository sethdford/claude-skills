# API Development Plugin

API design and development from REST/GraphQL/gRPC architecture through versioning, documentation, and testing.

## Skills

1. **rest-api-design** - RESTful principles, resource modeling, HTTP semantics
2. **graphql-schema-design** - Type systems, queries, mutations, subscriptions
3. **grpc-service-design** - Protocol buffers, service definitions, streaming
4. **api-versioning-strategy** - Backward compatibility, deprecation, versioning patterns
5. **api-documentation** - OpenAPI/Swagger, schema-driven docs, examples
6. **rate-limiting-design** - Token bucket, sliding window, quota strategies
7. **api-error-handling** - Status codes, error formats, recovery guidance
8. **api-testing-strategy** - Contract testing, API mocking, integration testing

## Commands

1. **design-api** - Complete REST/GraphQL API design with schema
2. **document-api** - Generate API documentation from schema
3. **test-api** - Design API testing strategy and contracts
