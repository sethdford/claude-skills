# DevOps Practices Plugin

Infrastructure, deployment, and operational excellence through CI/CD, containerization, and monitoring.

## Skills

1. **ci-cd-pipeline-design** - GitHub Actions, GitLab CI, Jenkins; stages, artifacts
2. **dockerfile-best-practices** - Container best practices, layer optimization, security
3. **deployment-strategy** - Blue-green, canary, rolling deploys
4. **monitoring-instrumentation** - Metrics, logs, traces, observability
5. **feature-flag-design** - Gradual rollouts, A/B testing, kill switches
6. **infrastructure-as-code** - Terraform, CloudFormation, reproducible infrastructure
7. **log-aggregation** - ELK, Datadog, structured logging, querying
8. **alerting-strategy** - SLOs, thresholds, incident escalation, runbooks

## Commands

1. **setup-pipeline** - Design CI/CD pipeline for project
2. **containerize** - Create Dockerfile following best practices
3. **deploy-strategy** - Plan safe deployment with blue-green or canary
