# Database Engineering Plugin

Database design and optimization from schema to production performance.

## Skills

1. **schema-design** - Normalization, relationships, constraints, data integrity
2. **query-optimization** - Execution plans, index selection, join strategies
3. **migration-strategy** - Safe schema changes, backward compatibility, rollback
4. **index-design** - B-tree indexes, composite indexes, covering indexes
5. **transaction-management** - ACID properties, isolation levels, locks
6. **data-integrity-patterns** - Constraints, triggers, audit trails
7. **connection-pool-management** - Pool sizing, connection lifetime, health checks

## Commands

1. **optimize-queries** - Analyze slow queries, add indexes, rewrite
2. **design-schema** - Design database schema for new feature
3. **plan-migration** - Safe schema migration with rollback plan

## Standards Reference

- C.J. Date, _An Introduction to Database Systems_ (8th edition)
- Joe Celko, _SQL Antipatterns_
