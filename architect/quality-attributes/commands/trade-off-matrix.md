# Trade-Off Matrix

Chain these steps:

1. Use `trade-off-analysis` to define criteria and weights.
2. Score each option on scalability, cost, deployability, security, reliability, maintainability.
3. Calculate weighted scores.
4. Document qualitative factors and risk assessment.

Produce: decision matrix with recommendation and trade-off rationale.
