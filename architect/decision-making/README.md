# Decision-Making Plugin

Master architectural decision-making through ADRs, technology selection, build-vs-buy analysis, migration planning, and risk assessment.

## Skills

- **architecture-decision-record** — Document decisions with ADR format for organizational memory
- **technology-radar** — Track and evaluate technologies on radar (Adopt/Trial/Assess/Hold)
- **build-vs-buy** — Compare custom building vs buying off-the-shelf solutions
- **proof-of-concept-plan** — Design focused POCs to validate assumptions
- **migration-strategy** — Plan phased migrations with minimal risk
- **legacy-modernization** — Modernize old systems incrementally
- **vendor-evaluation** — Systematically evaluate and select vendors
- **risk-assessment** — Identify, prioritize, and mitigate architectural risks

## Commands

### make-decision

Document and justify an architectural decision using ADR format and trade-off analysis.

### evaluate-technology

Assess technology and position on radar; create evaluation report.

### plan-migration

Develop phased migration strategy with risk mitigation and success metrics.
