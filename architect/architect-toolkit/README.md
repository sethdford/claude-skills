# Architect Toolkit Plugin

Cross-cutting utilities for architects: katas, interview prep, review facilitation, mentoring, anti-pattern recognition, audits, and pattern catalogs.

## Skills

- **architecture-kata** — Practice architectural design with structured problems
- **interview-prep** — Prepare for architecture interview questions and scenarios
- **architecture-review-facilitation** — Lead effective architecture reviews
- **technical-mentoring** — Coach engineers on architecture and design
- **architecture-anti-patterns** — Identify and avoid common architectural mistakes
- **system-audit** — Comprehensive system architecture evaluation
- **architecture-patterns-catalog** — Reference catalog of proven patterns

## Commands

### run-kata

Execute architecture kata with time-limited design challenge.

### audit-system

Conduct comprehensive system architecture audit with recommendations.

### facilitate-review

Lead architecture review session with structured approach.

## When to Use

- Practicing architecture design skills
- Preparing for architecture interviews
- Leading architecture reviews
- Mentoring architects and senior engineers
- Identifying anti-patterns and improvements
- Conducting system audits
- Referencing proven architecture patterns
