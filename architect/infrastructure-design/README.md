# Infrastructure Design Plugin

Master cloud architecture, deployment topology, infrastructure-as-code, and disaster recovery for scalable, resilient systems.

## Skills

- **cloud-architecture** — Design cloud infrastructure for AWS, GCP, or Azure
- **deployment-topology** — Plan container orchestration, load balancing, HA
- **infrastructure-as-code-patterns** — Code your infrastructure with Terraform, CloudFormation, Pulumi
- **disaster-recovery-plan** — Define RTO/RPO, backup strategies, failover procedures
- **multi-region-design** — Distribute globally with low latency and data locality
- **container-orchestration** — Kubernetes scaling, service discovery, networking
- **network-topology** — Design VPCs, security groups, load balancing, DNS
- **capacity-planning** — Forecast resource needs and utilization targets

## Commands

### design-infrastructure

Design complete cloud infrastructure for new system.

### plan-deployment

Create deployment topology with HA, scaling, and orchestration.

### disaster-recovery

Plan recovery strategy with RTO, RPO, and failover procedures.

## When to Use

- Designing new cloud infrastructure
- Planning Kubernetes deployments
- Establishing disaster recovery
- Multi-region architecture design
- Capacity planning and cost optimization
- Infrastructure-as-code automation
