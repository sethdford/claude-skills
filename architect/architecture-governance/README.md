# Architecture Governance Plugin

Establish architectural principles, fitness functions, and governance practices. Manage tech debt and ensure compliance.

## Skills

- **architecture-principles** — Define and communicate foundational architecture principles
- **fitness-function-design** — Design metrics that measure architecture health
- **tech-debt-assessment** — Measure, prioritize, and address technical debt
- **architecture-review-checklist** — Standardized architecture review process
- **compliance-framework** — Implement compliance requirements (SOC2, GDPR, HIPAA)
- **dependency-governance** — Manage and optimize system dependencies
- **change-impact-analysis** — Predict impacts of architectural changes

## Commands

### run-review

Execute architecture review using standardized checklist and fitness functions.

### assess-debt

Evaluate technical debt and create paydown roadmap.

### define-principles

Establish architecture principles and communicate organization-wide.

## When to Use

- Establishing architecture governance practices
- Defining architectural standards and principles
- Technical debt management and prioritization
- Compliance and regulatory requirement implementation
- Architecture review and quality gates
- Managing cross-system dependencies
