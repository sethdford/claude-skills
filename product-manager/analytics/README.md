# Analytics Plugin

Metrics, experimentation, and data-informed product decisions.

## Skills (8)

1. **metrics-framework** — Define comprehensive metrics tree
2. **experiment-design** — Design statistically sound experiments
3. **funnel-analysis** — Analyze conversion funnels
4. **cohort-analysis** — Track cohort behavior over time
5. **retention-analysis** — Measure and improve retention
6. **a-b-test-design-pm** — Plan A/B tests that isolate causation
7. **data-storytelling** — Transform data into compelling narratives
8. **north-star-metric** — Define single north-star metric

## Commands (3)

1. **define-metrics** — Define comprehensive metrics framework
2. **design-experiment** — Design A/B test
3. **analyze-funnel** — Analyze conversion funnel

## Key Concepts

- **North star metric**: Single metric that drives all decisions
- **Leading indicators**: Early signals of future performance
- **Experimentation velocity**: Fast learning beats big bets
- **Data storytelling**: Metrics only matter if communicated effectively
- **Cohort tracking**: Understand segment-specific behavior
