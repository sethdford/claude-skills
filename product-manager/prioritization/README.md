# Prioritization Plugin

Prioritization frameworks, roadmapping, and trade-off decision-making.

## Skills (8)

1. **rice-scoring** — Score opportunities with Reach, Impact, Confidence, Effort
2. **impact-effort-matrix** — Visualize by impact and effort to find quick wins
3. **opportunity-scoring** — Find whitespace using Importance/Satisfaction/Market Size
4. **moscow-prioritization** — Categorize as Must/Should/Could/Won't-haves
5. **kano-analysis** — Classify features as Basic/Performance/Delighter
6. **weighted-scoring** — Build custom frameworks aligned to strategy
7. **cost-of-delay** — Quantify business cost of delayed features
8. **roadmap-design** — Structure transparent strategic roadmap

## Commands (3)

1. **prioritize-backlog** — Score and rank backlog items
2. **build-roadmap** — Create transparent strategic roadmap
3. **evaluate-trade-offs** — Analyze trade-offs between competing priorities

## Key Concepts

- **Quantitative prioritization**: Remove bias, enable comparison
- **Multiple frameworks**: Use 2-3 frameworks to validate rankings
- **Quick wins first**: Prioritize high impact/low effort
- **Strategic bets**: Also schedule high impact/high effort strategic work
- **Transparent roadmap**: Clear what's committed vs. exploring
