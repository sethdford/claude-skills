# RICE Scoring: Opportunity Prioritization [Quarter/Period]

## Scoring Framework

**Definition of RICE parameters** (team calibration):

- **Reach**: # of users affected per quarter (not total users)
- **Impact**: Value per affected user (3=massive/transforms workflow, 2=high/meaningful, 1=medium/nice-to-have, 0.5=low/minor polish)
- **Confidence**: Estimate accuracy (100%=measured/shipped before, 75%=good data, 50%=educated guess, 25%=speculative)
- **Effort**: Engineering labor in person-weeks (including design, QA, deployment)
- **Formula**: (Reach × Impact × Confidence) / Effort = RICE Score
