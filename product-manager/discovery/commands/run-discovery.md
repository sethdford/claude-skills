description: Execute a complete discovery sprint to identify and validate problems.
argument-hint: [problem area or opportunity to explore]
---

Chain these steps:

1. Use the `problem-statement` skill to define the core problem
2. Use the `customer-interview-guide` skill to prepare for 5-8 customer interviews
3. Use the `voice-of-customer` skill to synthesize interview findings
4. Use the `opportunity-assessment` skill to evaluate market size and addressability
5. Use the `assumption-mapping` skill to identify what must be true for this opportunity

After completion, suggest follow-up commands: `interview-customers`, `size-opportunity`, `map-assumptions`
