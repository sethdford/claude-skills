# Launch Plugin

Launch planning, go-to-market strategy, and rollout execution.

## Skills (8)

1. **launch-plan** — Create comprehensive launch plan
2. **go-to-market-strategy** — Develop GTM that reaches target customers
3. **beta-program-design** — Design and execute beta programs
4. **release-notes** — Write clear, user-focused release notes
5. **rollout-strategy** — Plan controlled rollout with monitoring
6. **launch-readiness-checklist** — Pre-launch alignment checklist
7. **feature-flag-strategy** — Use flags for safe continuous deployment
8. **success-criteria** — Define launch success metrics

## Commands (3)

1. **plan-launch** — Create comprehensive launch plan
2. **write-release-notes** — Write release notes
3. **assess-readiness** — Assess cross-functional readiness

## Key Concepts

- **Coordinated launch**: Product, marketing, sales, ops alignment
- **Clear messaging**: Value communicated clearly to customers
- **Controlled rollout**: Feature flags enable safe deployment
- **Monitoring**: Watch metrics closely post-launch
- **Beta feedback**: Gather real feedback before broad launch
