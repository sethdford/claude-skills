# Stakeholder Management Plugin

Stakeholder communication, alignment, and cross-functional coordination.

## Skills (7)

1. **stakeholder-map** — Identify stakeholders and map interests/influence
2. **executive-summary** — Write concise executive summaries
3. **status-update** — Create effective status updates
4. **decision-log** — Document decisions and rationale
5. **cross-functional-alignment** — Align teams around strategy
6. **feedback-synthesis** — Aggregate feedback into insights
7. **change-management** — Plan product changes for organizational alignment

## Commands (3)

1. **write-update** — Write effective status updates
2. **align-stakeholders** — Align teams and stakeholders
3. **log-decisions** — Document key decisions

## Key Concepts

- **Clear communication**: Tailor messaging to audience
- **Transparency**: Share rationale, not just decisions
- **Listening**: Stakeholder feedback improves decisions
- **Documentation**: Decisions recorded prevent misunderstanding
- **Alignment**: Everyone understands and supports direction
