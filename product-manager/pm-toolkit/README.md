# PM Toolkit Plugin

Cross-cutting PM utilities and essential PM practices.

## Skills (7)

1. **product-review** — Conduct effective product reviews
2. **retrospective-facilitation** — Run retrospectives that drive improvement
3. **one-pager** — Create compelling one-pagers
4. **product-brief** — Write tight product briefs
5. **customer-advisory-board** — Design and run advisory boards
6. **product-ops-playbook** — Build operational playbooks
7. **pm-interview-prep** — Prepare for PM interviews

## Commands (3)

1. **run-retro** — Facilitate effective retrospectives
2. **write-brief** — Write product brief
3. **product-review** — Conduct product review

## Key Concepts

- **Structured practices**: Repeatable frameworks are more effective
- **Facilitation skills**: Guide discussions; don't dominate
- **Documentation**: Record decisions to prevent rework
- **Customer connection**: Direct contact keeps product grounded
- **Continuous improvement**: Regular retrospectives identify improvements
