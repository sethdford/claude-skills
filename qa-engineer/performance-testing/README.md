# Performance Testing Plugin

Load testing, stress testing, and performance analysis.

## Overview

This plugin provides guidance for:
- Load test planning and design
- Stress testing and breaking point analysis
- Endurance testing for stability
- Spike testing for sudden load changes
- Performance baseline establishment
- Bottleneck identification and analysis
- Capacity planning from QA perspective
- Performance result reporting

## Skills (8)

| Skill | Purpose |
|-------|---------|
| `load-test-design` | Load test planning and design |
| `stress-test-plan` | Stress test strategy |
| `endurance-test-plan` | Long-running performance testing |
| `spike-test-plan` | Spike load testing |
| `performance-baseline` | Establishing baselines |
| `bottleneck-analysis` | Performance bottleneck identification |
| `capacity-planning-qa` | Capacity planning from QA perspective |
| `performance-test-report` | Analysis and reporting |

## Commands (3)

| Command | Purpose |
|---------|---------|
| `plan-load-test` | Design load test strategy |
| `analyze-performance` | Analyze performance results |
| `report-results` | Create performance report |

## When to Use

- Planning load tests for new releases
- Analyzing performance bottlenecks
- Establishing performance baselines
- Capacity planning and forecasting
- Stress testing for stability validation
- Performance regression detection
