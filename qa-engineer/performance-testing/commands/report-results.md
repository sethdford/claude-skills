# report-results

Create performance analysis and findings report

## Chains

1. **performance-test-report** — Generate report
2. **bottleneck-analysis** — Explain findings
3. **capacity-planning-qa** — Recommend capacity

## Usage

```
/report-results "Describe your testing need"
```

## Expected Output

- Comprehensive analysis or plan
- Actionable recommendations
- Implementation steps
- Success criteria
