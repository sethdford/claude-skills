# Automation Plugin

Test automation architecture, frameworks, and maintenance strategies.

## Overview

This plugin provides guidance for:
- Test automation architecture and design patterns
- Framework selection and evaluation
- Page object model implementation
- API test automation
- Data-driven and keyword-driven testing
- Visual regression testing
- Test maintenance and evolution
- Flaky test diagnosis and remediation

## Skills (9)

| Skill | Purpose |
|-------|---------|
| `automation-architecture` | Architecture and design patterns |
| `test-framework-selection` | Framework evaluation and selection |
| `page-object-model` | POM implementation |
| `api-test-automation` | API automation techniques |
| `data-driven-testing` | Parameterized test design |
| `keyword-driven-testing` | Keyword-driven frameworks |
| `visual-regression-testing` | Visual regression strategies |
| `test-maintenance-strategy` | Maintaining automation suites |
| `flaky-test-remediation` | Fixing flaky tests |

## Commands (4)

| Command | Purpose |
|---------|---------|
| `design-automation` | Design automation strategy |
| `setup-framework` | Select and configure framework |
| `fix-flaky-tests` | Diagnose and remediate flaky tests |
| `automate-suite` | Automate existing test cases |

## When to Use

- Building test automation from scratch
- Selecting test automation frameworks
- Implementing page object patterns
- Maintaining large automation suites
- Fixing flaky or unreliable tests
- Designing data-driven test execution

- Test Automation Engineering
