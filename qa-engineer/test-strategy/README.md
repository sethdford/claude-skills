# Test Strategy Plugin

Test planning, risk-based testing, coverage analysis, and test environment management.

## Overview

This plugin provides strategic guidance for:

- Test strategy and planning
- Risk-based test prioritization
- Coverage measurement and analysis
- Test pyramid architecture
- Shift-left quality practices
- Regression testing strategy
- Test environment design
- Test data management
- Testing effort estimation

## Skills (9)

| Skill                    | Purpose                            |
| ------------------------ | ---------------------------------- |
| `test-strategy-design`   | Strategic test planning frameworks |
| `risk-based-test-plan`   | Risk-based test prioritization     |
| `test-coverage-analysis` | Coverage measurement and gaps      |
| `test-pyramid-design`    | Unit/integration/E2E balance       |
| `shift-left-strategy`    | Early testing approaches           |
| `regression-strategy`    | Regression test management         |
| `test-environment-plan`  | Environment strategy and setup     |
| `test-data-strategy`     | Test data management               |
| `test-estimation`        | Test effort estimation             |

## Commands (4)

| Command           | Purpose                            |
| ----------------- | ---------------------------------- |
| `plan-testing`    | Design comprehensive test strategy |
| `assess-coverage` | Analyze test coverage gaps         |
| `design-pyramid`  | Design test pyramid architecture   |

## When to Use

- Planning test efforts for new projects or releases
- Designing test strategies aligned with project goals
- Analyzing coverage gaps and planning coverage improvements
- Architecting test automation at different levels
- Estimating testing resources and timelines
- Implementing shift-left quality practices
- Managing regression test suites
- Designing test environments
- Planning test data provisioning
