# Quality Metrics Plugin

Quality measurement, defect analysis, and process improvement.

## Overview

This plugin provides guidance for:
- Defect analysis and trend analysis
- Quality dashboard design
- Test metrics framework definition
- Defect prevention strategies
- Root cause analysis for quality issues
- Quality gate design and implementation
- Escaped defect analysis
- Test effectiveness measurement

## Skills (8)

| Skill | Purpose |
|-------|---------|
| `defect-analysis` | Defect analysis and trends |
| `quality-dashboard` | Metrics dashboard design |
| `test-metrics-framework` | Test metrics definition |
| `defect-prevention` | Root cause prevention |
| `root-cause-analysis-qa` | RCA for quality issues |
| `quality-gate-design` | Quality gate definition |
| `escaped-defect-analysis` | Post-release defect analysis |
| `test-effectiveness-measurement` | Testing ROI measurement |

## Commands (3)

| Command | Purpose |
|---------|---------|
| `analyze-defects` | Analyze defect data and trends |
| `build-dashboard` | Design quality dashboard |
| `assess-quality-gates` | Evaluate quality gate effectiveness |

## When to Use

- Measuring and tracking quality metrics
- Analyzing defect patterns and trends
- Designing quality dashboards
- Establishing quality gates
- Analyzing escaped defects from production
- Measuring testing effectiveness

## Standards Grounding

- ISO/IEC 25010 Software Quality Model
- Google Testing Blog — Quality measurement
- Data-driven quality management
