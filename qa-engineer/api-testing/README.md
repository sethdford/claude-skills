# API Testing Plugin

API validation, contract testing, and integration verification.

## Overview

This plugin provides guidance for:
- API testing strategy and planning
- Consumer-driven contract testing
- Schema and payload validation
- API security testing
- API performance testing
- API mocking and stubbing strategies
- API regression test suite development

## Skills (7)

| Skill | Purpose |
|-------|---------|
| `api-test-plan` | API testing strategy |
| `contract-testing` — | Consumer-driven contract testing |
| `schema-validation` | Schema and payload validation |
| `api-security-testing` | API security testing |
| `api-performance-testing` | API performance analysis |
| `api-mock-strategy` | Mocking and stubbing strategies |
| `api-regression-suite` | Building API regression suites |

## Commands (3)

| Command | Purpose |
|---------|---------|
| `test-api` | Design API test suite |
| `validate-contracts` | Set up contract testing |
| `setup-mocks` | Configure API mocks and stubs |

## When to Use

- Testing APIs and integration endpoints
- Validating API contracts between consumers and providers
- Testing API security vulnerabilities
- Setting up mock APIs for isolated testing
- Building regression suites for API stability

## Standards Grounding

- API testing best practices
- Contract testing patterns
- RESTful API design principles
