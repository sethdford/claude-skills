# explore-feature

Plan and execute exploratory testing session

## Chains

1. **exploratory-testing-charter** — Define exploration mission
2. **error-guessing** — Discover unexpected behaviors
3. **test-case-design** — Document findings as test cases

## Usage

```
/explore-feature "Describe your testing need"
```

## Expected Output

- Comprehensive analysis or plan
- Actionable recommendations
- Implementation steps
- Success criteria
