# QA Toolkit Plugin

Cross-cutting QA utilities and operational excellence.

## Overview

This plugin provides guidance for:
- Effective bug reporting
- Test plan review and validation
- QA team onboarding
- Test environment management
- QA retrospectives
- Release readiness assessment
- QA knowledge base building

## Skills (7)

| Skill | Purpose |
|-------|---------|
| `bug-report-template` | Effective bug reporting |
| `test-plan-review` | Test plan review checklist |
| `qa-onboarding` | QA team onboarding |
| `test-environment-management` | Environment management |
| `qa-retrospective` | QA team retrospectives |
| `release-readiness-assessment` | Release quality gates |
| `qa-knowledge-base` | Building QA knowledge base |

## Commands (3)

| Command | Purpose |
|---------|---------|
| `review-test-plan` | Conduct test plan review |
| `assess-release` | Assess release readiness |
| `run-qa-retro` | Facilitate QA retrospective |

## When to Use

- Reviewing test plans
- Assessing release readiness
- Onboarding new QA team members
- Managing test environments
- Running QA retrospectives
- Building team knowledge base

## Standards Grounding

- QA best practices
- Team process improvement
- Operational excellence
