description: Systematically evaluate and recommend technology choices.
argument-hint: [problem domain, candidate technologies, current stack]
---

Chain these steps:

1. Use the `technology-evaluation` skill to define evaluation criteria
2. Use the `decision-matrix` skill to score options against criteria
3. Use the `spike-plan` skill to design proof-of-concept if needed
4. Use the `build-vs-buy-analysis` skill to evaluate custom vs. off-the-shelf
5. Synthesize into a recommendation with trade-off analysis and risk assessment

After completion, suggest:

- Migration or implementation plan if new technology is chosen
- Training needs for team
- How to measure success post-adoption
