---
description: Write a comprehensive RFC that drives collaborative technical decisions.
argument-hint: [decision scope, stakeholders, timeline]
---

Chain these steps:

1. Use the `rfc-process` skill to structure the RFC and define review timeline
2. Use the `decision-matrix` skill to compare options quantitatively
3. Use the `risk-mitigation-plan` skill to identify risks in chosen option
4. Use the `post-decision-review` skill to plan how you'll measure success
5. Synthesize into an RFC document with options, trade-offs, recommendation, and risks

After completion, suggest:

- How to socialize the RFC before formal review
- How to gather stakeholder feedback
- When and how to finalize the decision
