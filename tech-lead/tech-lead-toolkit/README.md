# Tech Lead Toolkit Plugin

Essential utilities and practices for effective tech leadership.

## Skills (7 skills)

- **tech-lead-journal** — Keep structured notes to track decisions, learnings, and team dynamics
- **team-health-check** — Regularly assess team health across engagement, velocity, and psychological safety
- **retrospective-facilitation-tl** — Run retrospectives that surface improvement opportunities without blame
- **engineering-blog-post** — Write technical blog posts that build team and company reputation
- **tech-talk-preparation** — Prepare and deliver talks on technical topics to build expertise visibility
- **hiring-interview-design** — Design technical interviews that assess capabilities fairly
- **team-charter** — Define team working agreements, values, and norms

## Commands (3 commands)

- **run-health-check** — Assess team health and identify gaps
- **facilitate-retro** — Run effective retrospective with clear outcomes
- **write-charter** — Create team charter with norms and values

## Key Principles

1. Structured reflection improves decision-making: journaling captures insights
2. Team health is measurable: engagement, velocity, and psychological safety can be tracked
3. Retrospectives drive improvement: focus on what's controllable, not blame
4. Visibility builds credibility: writing and speaking about technical work builds influence
5. Clear norms reduce friction: team charters set expectations upfront
