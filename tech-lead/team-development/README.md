# Team Development Plugin

Build high-performing teams through mentoring, growth planning, and career development.

## Skills (8 skills)

- **one-on-one-template** — Structure effective 1:1s that build trust and surface concerns
- **mentoring-plan** — Create structured mentoring relationships with clear goals and milestones
- **skill-matrix** — Map team capabilities to identify skill gaps and development opportunities
- **career-ladder-design** — Design career progression frameworks that clarify growth paths
- **pair-programming-facilitation** — Run effective pairing sessions for knowledge transfer and learning
- **knowledge-sharing-plan** — Systematize knowledge transfer through brown bags, docs, and mentoring
- **onboarding-program** — Design onboarding that accelerates time-to-productivity for new hires
- **performance-feedback** — Deliver feedback that improves performance and builds trust

## Commands (3 commands)

- **plan-mentoring** — Create mentoring relationship with goals and structure
- **build-skill-matrix** — Map team skills and identify development opportunities
- **design-onboarding** — Build comprehensive onboarding program

## Key Principles

1. Growth happens through psychological safety and clear expectations
2. Career ladders reduce ambiguity about advancement
3. Knowledge transfer must be systematic, not accidental
4. Pair programming is a teaching tool, not a code review replacement
5. Great mentoring unblocks individuals and multiplies team output
