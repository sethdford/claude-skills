# SDLC Cross-Role Skills

**8 skills | 8 commands | 1 plugin**

Standards: ISO/IEC 12207, ISO/IEC 15288, CMMI v2.0, SAFe 6.0, DORA/Accelerate.

Cross-functional workflows that chain skills from 3-5 roles into unified lifecycle processes.

## Skills

| Skill                   | Focus                                            |
| ----------------------- | ------------------------------------------------ |
| sdlc-phase-gate         | Quality gates between SDLC phases                |
| definition-of-ready     | Readiness criteria for work items                |
| definition-of-done      | Completion criteria across roles                 |
| feature-lifecycle       | End-to-end feature delivery workflow             |
| release-checklist       | Cross-functional release validation              |
| incident-to-improvement | Post-incident learning loops                     |
| retrospective-to-action | Turning retro insights into tracked improvements |
| compliance-traceability | Requirements-to-deployment audit trail           |

## Commands

| Command               | Focus                                     |
| --------------------- | ----------------------------------------- |
| /design-review        | Architecture + security + UX review       |
| /feature-kickoff      | PM + architect + tech lead alignment      |
| /pre-launch-review    | Cross-role launch readiness check         |
| /security-review      | Threat model + secure code + infra review |
| /quality-gate         | Multi-role phase transition validation    |
| /full-lifecycle       | Complete feature delivery orchestration   |
| /post-incident-review | Incident -> RCA -> improvement pipeline   |
| /sprint-ceremony      | Sprint planning + review + retro          |
